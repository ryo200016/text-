## 下記for文の`let i = 0`, `i < 100`, `$i++`, `console.log(i)`がそれぞれ、どの順番で処理されるか、また、何をしているのかを説明してください。

```
for (let i = 0; i < 100; i++) {
  console.log(i);
}
```

1. let i でiの初期値に０を代入。
2. i < 100　100より小さいか確認する。
3. console.log(i)　iに代入された値をlogに出力
4. i++　iの値を1増やす。
